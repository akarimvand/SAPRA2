# 05 - راهنمای کاربری

## ورود به سیستم

### صفحه ورود (index.html)
1. انتخاب فاز پروژه از dropdown
2. وارد کردن نام کاربری
3. وارد کردن رمز عبور
4. کلیک روی "ورود به سیستم"

## داشبورد اصلی

### نوار کناری (Sidebar)
- **All Systems**: نمایش تمام سیستمها
- **سیستمها**: لیست سیستمهای پروژه
- **زیرسیستمها**: زیرمجموعه هر سیستم
- **جستجو**: جعبه جستجو برای فیلتر کردن

### کارتهای خلاصه
1. **Total Items**: تعداد کل آیتمهای انتخاب شده
2. **Completed**: آیتمهای تکمیل شده (سبز)
3. **Pending**: آیتمهای در انتظار (زرد)
4. **Remaining**: آیتمهای باقیمانده (آبی)
5. **Issues**: پانچها و Hold Pointها (قرمز)

### کارتهای فرم
1. **Total Forms**: تعداد کل زیرسیستمها
2. **Form A**: فرمهای ارسال شده به کلاینت
3. **Form B**: فرمهای بازگشتی با پانچ
4. **Form C**: فرمهای پاکسازی شده
5. **Form D**: فرمهای تایید نهایی

## تبهای اصلی

### Home Tab
- کارتهای خلاصه
- چارتهای تحلیلی
- جدول جزئیات

### Charts Tab
#### Overview Charts
- نمودار دونات کلی وضعیت پروژه
- نمایش درصد تکمیل

#### By Discipline Charts
- نمودار جداگانه برای هر رشته
- مقایسه عملکرد رشتهها

### Workflow Tab
- iframe مربوط به گردش کار
- نمایش فرآیند تحویل

### Reports Tab
- گزارش پانچها
- نمودار روند بسته شدن پانچها
- امکان صادرات

## جدول جزئیات

### ستونهای جدول
- **System**: نام سیستم
- **Subsystem**: نام زیرسیستم
- **Form**: وضعیت فرم (A, B, C, D)
- **Discipline**: رشته مهندسی
- **Total Items**: تعداد کل آیتمها
- **Completed**: تعداد تکمیل شده
- **Pending**: تعداد در انتظار
- **Punch**: تعداد پانچها
- **Hold Point**: تعداد نقاط نگهداری
- **Status**: درصد پیشرفت

### فیلترهای جدول
- **فیلتر متنی**: برای تمام ستونها
- **فیلتر Dropdown**: برای Discipline و Form
- **دکمه Clear All**: پاک کردن تمام فیلترها

## مودالهای جزئیات

### کلیک روی کارتها
- کلیک روی هر کارت نمایش جزئیات مربوطه
- فیلتر خودکار بر اساس انتخاب

### کلیک روی جدول
- کلیک روی هر سلول نمایش آیتمهای مربوطه
- جزئیات کامل آیتمها

### مودال Activities
- کلیک روی Tag No
- نمایش فعالیتهای مربوط به آیتم
- نوار پیشرفت فعالیتها

## ویژگیهای جستجو

### جستجوی سریع (Ctrl+F)
- جستجو در Tag No، Description، Subsystem
- نمایش نتایج با امکان کلیک
- فیلتر خودکار به زیرسیستم

### جستجوی sidebar
- فیلتر سیستمها و زیرسیستمها
- نمایش فقط موارد مطابق

## صادرات و گزارشگیری

### صادرات Excel
- **Export Excel**: صادرات جدول اصلی
- **Export Details**: صادرات جزئیات مودال
- **Export Activities**: صادرات فعالیتهای آیتم

### چاپ
- **Print Table**: چاپ جدول اصلی
- **Print Modal**: چاپ محتوای مودال

### دانلود همه فایلها
- دانلود تمام فایلهای CSV در یک ZIP
- شامل تمام دادههای پروژه

## میانبرهای کیبورد

- **Ctrl+E**: صادرات
- **Ctrl+R**: رفرش دادهها
- **Ctrl+F**: جستجوی سریع
- **Ctrl+D**: دیتابیس
- **Ctrl+Q**: خروج
- **?**: نمایش میانبرها

## نمای موبایل

### نوار پایین
- **Home**: صفحه اصلی
- **Charts**: نمودارها
- **Workflow**: گردش کار
- **Reports**: گزارشات
- **Menu**: منوی اضافی

### کارتهای موبایل
- نمایش دادهها در قالب کارت
- امکان کلیک برای جزئیات
- جستجوی موبایل

## نکات کاربردی

### بهینهسازی عملکرد
- استفاده از فیلترها برای کاهش دادهها
- بستن مودالهای غیرضروری
- رفرش دادهها در صورت نیاز

### عیبیابی
- بررسی اتصال اینترنت
- رفرش صفحه (Ctrl+F5)
- بررسی console مرورگر
