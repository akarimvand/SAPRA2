# SAPRA - Smart Access to Project Activities

## نمای کلی
SAPRA یک سیستم مدیریت پروژه هوشمند است که برای نظارت و کنترل فعالیت‌های پروژه‌های صنعتی طراحی شده است. این سیستم امکان ردیابی وضعیت تکمیل آیتم‌ها، مدیریت پانچ‌ها و نظارت بر فرآیند تحویل زیرسیستم‌ها را فراهم می‌کند.

## ویژگی‌های کلیدی

### 📊 داشبورد تحلیلی
- نمایش آمار کلی پروژه (تکمیل شده، در انتظار، باقیمانده)
- چارت‌های تعاملی با amCharts
- فیلتر بر اساس سیستم و زیرسیستم
- نمایش وضعیت فرم‌های A، B، C، D

### 🔍 جستجو و فیلتر پیشرفته
- جستجوی سریع آیتم‌ها
- فیلتر بر اساس رشته، وضعیت، زیرسیستم
- صادرات داده‌های فیلتر شده به Excel

### 📈 گزارش‌گیری
- گزارش وضعیت پانچ‌ها
- نمودار روند بسته شدن پانچ‌ها
- صادرات تصاویر و Excel

### 🔄 مدیریت گردش کار
- ردیابی مراحل تحویل زیرسیستم‌ها
- مدیریت فرم‌های تحویل
- نمایش وضعیت Hold Point ها

## ساختار فایل‌ها

```
SAPRA2-main/
├── index.html          # صفحه ورود
├── main.html          # داشبورد اصلی
├── main.js            # منطق اصلی برنامه
├── main.css           # استایل‌های اصلی
├── reports.html       # گزارش پانچ‌ها
├── dbcsv/            # فایل‌های داده CSV
├── amcharts/         # کتابخانه چارت محلی
├── images/           # تصاویر و لوگو
└── pages/            # صفحات اضافی
```

## نحوه استفاده

### 1. ورود به سیستم
- انتخاب فاز پروژه
- وارد کردن نام کاربری و رمز عبور
- دسترسی به داشبورد اصلی

### 2. مشاهده داشبورد
- انتخاب سیستم یا زیرسیستم از منوی کناری
- مشاهده آمار و چارت‌ها
- کلیک روی کارت‌ها برای جزئیات بیشتر

### 3. گزارش‌گیری
- رفتن به تب "Punch Release"
- انتخاب بازه زمانی و رشته‌ها
- صادرات گزارش‌ها

## فناوری‌های استفاده شده

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **UI Framework**: Bootstrap 5
- **Charts**: amCharts 5
- **Data Processing**: Papa Parse (CSV)
- **Export**: SheetJS (Excel), html2canvas (Images)
- **Icons**: Bootstrap Icons

## نصب و راه‌اندازی

1. فایل‌های پروژه را در وب سرور قرار دهید
2. فایل‌های amCharts را در پوشه `amcharts/` کپی کنید
3. فایل‌های CSV را در پوشه `dbcsv/` قرار دهید
4. به `index.html` مراجعه کنید
