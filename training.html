SAPRA Training Guide

Learn SAPRA System

Complete training guide for Smart Access to Project Activities system usage and features.

Getting Started

SAPRA is an intelligent project management system designed for monitoring and controlling activities in industrial projects. This guide will help you understand and effectively use all system features.

Step-by-Step Tutorial

1
System Login

Access the system through the login page, select your project phase, and enter your credentials.

2
Dashboard Overview

The main dashboard displays project statistics, form status cards, and interactive charts.

3
Navigation

Use the sidebar to select systems and subsystems. The tree view shows hierarchical project structure.

4
Data Analysis

Click on summary cards to view detailed information. Use charts for visual data analysis.

5
Reporting

Export data to Excel format and generate comprehensive project reports.

Key Features

  • Real-time Dashboard - Live project status updates
  • System Management - Hierarchical project organization
  • Interactive Charts - Visual data representation
  • Form Tracking - Monitor Forms A, B, C, D status
  • Issue Management - Track punches and hold points
  • Export Capabilities - Generate Excel reports

Tips & Best Practices

  • Click on Tag Numbers to view related activities
  • Use search and filter functions for quick data access
  • Hover over charts for detailed information
  • Use keyboard shortcuts for faster navigation

شروع کار

SAPRA یک سیستم مدیریت پروژه هوشمند است که برای نظارت و کنترل فعالیتهای پروژههای صنعتی طراحی شده است. این راهنما به شما کمک میکند تا تمام امکانات سیستم را درک کرده و به طور مؤثر استفاده کنید.
