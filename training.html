Main Dashboard

Comprehensive overview of project status including completed, pending, and remaining items with real-time updates.

System Management

Organize projects by systems and subsystems with advanced search and filtering capabilities.

Analytics Charts

Interactive visual data representation with doughnut charts for overall status and discipline breakdowns.

How It Works

1
System Initialization

The system automatically loads project data from online sources upon page load.

2
Select System or Subsystem

Choose your desired system or subsystem from the right sidebar menu.

3
Review Summary Information

Summary cards at the top display overall status. Click on any card to view details.

4
Analyze Charts

Use the "Overview" and "By Discipline" tabs to view analytical charts.

5
Explore Detailed Tables

The bottom table shows detailed information for each discipline.

Pro Tips

Essential Tips
  • Click on Tag Numbers to view related activities
  • Use table filters for quick data search
  • Charts are interactive with hover details
  • Data updates automatically from online sources
